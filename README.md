# 🔐 Cybersecurity Tools in Python

## 📌 Overview
This repository contains a collection of basic cybersecurity tools developed using Python.  
These tools demonstrate fundamental concepts of network security and system protection.

---

## ⚙️ Tools Included

### 1. Password Strength Checker
- Evaluates the strength of a password
- Checks for:
  - Minimum length
  - Uppercase letters
  - Numbers
  - Special characters

### 2. Port Scanner
- Scans open ports on a target system
- Helps identify potential vulnerabilities
- Uses socket programming

---

## 🛠️ Technologies Used
- Python
- Socket Programming
- Regular Expressions
